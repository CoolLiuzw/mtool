# mtool
- 一个简单的 vscode 格式化 json 日志的插件，支持多行的 json 解析。
- 解析json日志方便阅读


## 使用方式

1. 安装该 vscode 插件
2. 打开需要格式化的 json log 文件
3. ctrl+shift+p 输入 `Log Format` , 回车执行即可。
